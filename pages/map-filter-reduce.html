Map Problems

5 Real-World map() Problems

1
Extract names from objects
map
interface User { name: string; age: number; }
const users: User[] = [
  { name: "Alice", age: 25 }, { name: "Bob", age: 30 }, { name: "Charlie", age: 22 }
];
const names = users.map(u => u.name);
// ["Alice", "Bob", "Charlie"]
2
Convert prices USD → INR
map
const usdPrices = [10, 25, 99, 5.5];
const inrPrices = usdPrices.map(p => +(p * 83.5).toFixed(2));
// [835, 2087.5, 8266.5, 459.25]
3
Format dates
map
const dates = ["2024-01-15", "2024-06-20", "2024-12-31"];
const formatted = dates.map(d => {
  const date = new Date(d);
  return date.toLocaleDateString("en-IN", { day: "numeric", month: "short", year: "numeric" });
});
// ["15 Jan 2024", "20 Jun 2024", "31 Dec 2024"]
4
Transform API response
map
interface ApiUser { first_name: string; last_name: string; email_address: string; }
interface AppUser { fullName: string; email: string; }

const apiData: ApiUser[] = [
  { first_name: "Alice", last_name: "Smith", email_address: "a@b.com" }
];
const appUsers: AppUser[] = apiData.map(u => ({
  fullName: `${u.first_name} ${u.last_name}`,
  email: u.email_address,
}));
5
Create HTML from data
map
const items = ["Milk", "Bread", "Eggs"];
const html = items.map(i => `<li class="item">${i}</li>`).join("\n");
// <li class="item">Milk</li>
// <li class="item">Bread</li>
// <li class="item">Eggs</li>
Filter Problems

5 Real-World filter() Problems

1
Filter users by age
filter
const users = [
  { name: "Alice", age: 25 }, { name: "Bob", age: 17 },
  { name: "Charlie", age: 30 }, { name: "Dave", age: 15 }
];
const adults = users.filter(u => u.age >= 18);
// [{ name: "Alice", age: 25 }, { name: "Charlie", age: 30 }]
2
Remove null/undefined
filter
const mixed = ["hello", null, "world", undefined, "", "foo"];
const clean = mixed.filter((x): x is string => x != null && x !== "");
// ["hello", "world", "foo"]
// The type guard (x): x is string narrows the type!
3
Search by text
filter
const products = ["iPhone 15", "Samsung Galaxy", "iPad Pro", "Pixel 8"];
const query = "ip";
const results = products.filter(p => p.toLowerCase().includes(query.toLowerCase()));
// ["iPhone 15", "iPad Pro"]
4
Filter by price range
filter
interface Product { name: string; price: number; }
const products: Product[] = [
  { name: "Pen", price: 10 }, { name: "Book", price: 250 },
  { name: "Laptop", price: 50000 }, { name: "Mouse", price: 500 }
];
const affordable = products.filter(p => p.price >= 100 && p.price <= 1000);
// [{ name: "Book", price: 250 }, { name: "Mouse", price: 500 }]
5
Find duplicates
filter
const nums = [1, 2, 3, 2, 4, 1, 5, 3];
const dupes = nums.filter((n, i) => nums.indexOf(n) !== i);
// [2, 1, 3] — items that appear more than once

const unique = nums.filter((n, i) => nums.indexOf(n) === i);
// [1, 2, 3, 4, 5] — remove duplicates (or use [...new Set(nums)])
Reduce Problems

5 Real-World reduce() Problems

1
Sum all numbers
reduce
const nums = [10, 20, 30, 40];
const total = nums.reduce((sum, n) => sum + n, 0);
// 100
2
Count occurrences (word frequency)
reduce
const words = ["apple", "banana", "apple", "cherry", "banana", "apple"];
const freq = words.reduce((acc, word) => {
  acc[word] = (acc[word] || 0) + 1;
  return acc;
}, {} as Record<string, number>);
// { apple: 3, banana: 2, cherry: 1 }
3
Flatten nested arrays
reduce
const nested = [[1, 2], [3, 4], [5, 6]];
const flat = nested.reduce((acc, arr) => [...acc, ...arr], [] as number[]);
// [1, 2, 3, 4, 5, 6]
// Modern alternative: nested.flat()
4
Group by category
reduce
interface Product { name: string; category: string; }
const products: Product[] = [
  { name: "iPhone", category: "phones" },
  { name: "MacBook", category: "laptops" },
  { name: "Pixel", category: "phones" },
  { name: "ThinkPad", category: "laptops" },
];
const grouped = products.reduce((acc, p) => {
  if (!acc[p.category]) acc[p.category] = [];
  acc[p.category].push(p.name);
  return acc;
}, {} as Record<string, string[]>);
// { phones: ["iPhone", "Pixel"], laptops: ["MacBook", "ThinkPad"] }
// Modern alternative: Object.groupBy(products, p => p.category)
5
Build object from key-value pairs
reduce
const pairs: [string, number][] = [["a", 1], ["b", 2], ["c", 3]];
const obj = pairs.reduce((acc, [key, val]) => {
  acc[key] = val;
  return acc;
}, {} as Record<string, number>);
// { a: 1, b: 2, c: 3 }
// Modern alternative: Object.fromEntries(pairs)
Chaining

Chaining map + filter + reduce

!
The real power — combine all three
Chain
interface Order { product: string; price: number; qty: number; status: string; }
const orders: Order[] = [
  { product: "Laptop", price: 50000, qty: 1, status: "delivered" },
  { product: "Mouse", price: 500, qty: 2, status: "delivered" },
  { product: "Keyboard", price: 1500, qty: 1, status: "cancelled" },
  { product: "Monitor", price: 15000, qty: 1, status: "delivered" },
];

// "Total revenue from delivered orders"
const revenue = orders
  .filter(o => o.status === "delivered")        // keep delivered only
  .map(o => o.price * o.qty)                    // calculate each total
  .reduce((sum, total) => sum + total, 0);      // sum them up
// 66000

// "Names of expensive delivered products (>1000)"
const expensive = orders
  .filter(o => o.status === "delivered" && o.price > 1000)
  .map(o => o.product);
// ["Laptop", "Monitor"]

⚠ Common Beginner Mistakes

⚠
Mistake 1: Using map() when you don't need the return value 
// ❌ WRONG — map creates a new array, but you're ignoring it
users.map(u => console.log(u.name));

// ✅ CORRECT — use forEach for side effects
users.forEach(u => console.log(u.name));
⚠
Mistake 2: Forgetting return in reduce() 
// ❌ WRONG — accumulator becomes undefined!
const sum = nums.reduce((acc, n) => { acc + n; }, 0);

// ✅ CORRECT — return the new accumulator
const sum2 = nums.reduce((acc, n) => { return acc + n; }, 0);
// Or use arrow shorthand:
const sum3 = nums.reduce((acc, n) => acc + n, 0);
⚠
Mistake 3: Wrong initial value in reduce() 
// ❌ WRONG — initial value is a number, but you want an object!
// words.reduce((acc, w) => { acc[w] = 1; return acc; }, 0);

// ✅ CORRECT — match initial value type to desired result
const freq = words.reduce((acc, w) => {
  acc[w] = (acc[w] || 0) + 1;
  return acc;
}, {} as Record<string, number>);
⚠
Mistake 4: Mutating original array inside map() 
// ❌ WRONG — mutating the original objects!
const updated = users.map(u => { u.age += 1; return u; });
// Original users are ALSO modified!

// ✅ CORRECT — return NEW objects
const updated2 = users.map(u => ({ ...u, age: u.age + 1 }));
⚠
Mistake 5: filter() returns a NEW array — must use it! 
// ❌ WRONG — ignoring the returned array
users.filter(u => u.age >= 18);  // result thrown away!

// ✅ CORRECT — use the returned array
const adults = users.filter(u => u.age >= 18);

Cheatsheet

// MAP — transform each item
arr.map(x => x * 2)                    // [2, 4, 6]
arr.map(u => u.name)                   // extract field

// FILTER — keep matching items
arr.filter(x => x > 2)                // remove small
arr.filter((x): x is T => x != null)  // remove nulls (typed)

// REDUCE — accumulate into one value
arr.reduce((sum, n) => sum + n, 0)                // sum
arr.reduce((acc, x) => ({ ...acc, [x]: 1 }), {})  // build object

// CHAIN — combine for complex transforms
arr.filter(condition).map(transform).reduce(accumulate, init)
